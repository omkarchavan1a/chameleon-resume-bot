# STACK

## Frontend
- **HTML5**: semantic markup for UI
- **CSS3**: Vanilla CSS with custom variables for styling, responsive design
- **JavaScript**: Vanilla JS for DOM manipulation and state management
- **Libraries**:
  - `marked.min.js`: Markdown parsing for resume output
  - `html2pdf.js`: Client-side PDF generation

## Backend
- **Python**: 3.8+ (implied)
- **FastAPI**: modern, fast (high-performance) web framework for building APIs
- **Uvicorn**: ASGI web server implementation
- **Pydantic**: data validation using Python type annotations
- **OpenAI SDK**: Used as a client for the NVIDIA NIM API (compatible with OpenAI interface)

## Tools
- **NVIDIA NIM API**: Llama 3.3 70B used for generative AI logic
